# GutenKu Front

## Installation

``` bash
cp .env.dist .env
yarn install
```

## Run

``` bash
yarn dev
```

## Tests e2e

Prerequisites:

``` bash
apt-get install libgtk2.0-0 libgtk-3-0 libgbm-dev libnotify-dev libgconf-2-4 libnss3 libxss1 libasound2 libxtst6 xauth xvfb
```

Cypress:

``` bash
yarn add cypress --dev -g
```
