# GutenKu Api

## Installation

```bash
cp .env.dist .env
yarn install
```

Batch:

```bash 
pip3 install -r requirements.txt
```

MongoDB:

Using Docker:

```bash
docker-compose up -d
```

## Usage

Fetch and store books:

```bash
yarn setup
```

## Run

```bash
yarn dev
```

## Generate

Generate Haiku:

```bash
yarn extract
```

## Post

Send Instagram Post:

```bash
yarn post
```
