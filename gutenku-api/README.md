# GutenKu Api

## Installation

```bash
cp .env.dist .env
yarn install
```

Batch:

```bash
mkdir .cache
pip install python-dotenv
pip install pymongo
```

MongoDB:

```bash
docker-compose up -d
```

## Usage

Fetch and store books

```bash
yarn fetch
```

## Run

```bash
yarn dev
```

## Batch

Send Instagram Post:

```bash
yarn post
```
